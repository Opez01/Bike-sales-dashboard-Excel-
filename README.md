# Bike Sales Analysis Dashboard (Excel)

## Overview

This project analyzes bike sales data to understand customer purchasing behavior and key factors influencing sales performance. An interactive Excel dashboard was developed to present insights in a clear and visual format.

---

##  Tools Used

* Microsoft Excel
* Pivot Tables
* Data Cleaning Techniques
* Data Visualization (Charts & Dashboard)

---

##  Dashboard Features

The dashboard focuses on the following key areas:

* **Average Income per Purchase**
  Analyzes how customer income levels influence purchasing decisions

* **Customer Age Brackets**
  Examines sales distribution across different age groups

* **Customer Commute Distance**
  Evaluates how commuting distance affects bike purchases

---

##  Key Insights

* Customers with higher income levels tend to purchase bikes more frequently
* Middle-aged customers represent a significant portion of total sales
* Customers with shorter commute distances show higher purchase rates
